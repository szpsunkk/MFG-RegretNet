## 2. 核心模块实现思路

### 2.1 隐私拍卖机制模块 (`privacy_auction.py`)

**功能**：实现PAC机制、隐私成本函数、效用计算

```markdown
### 主要类和函数

#### PrivacyAuctionGame
- `__init__(N, m, B, valuation_dist)`: 初始化拍卖游戏
  - N: 客户端数量
  - m: 隐私项目数
  - B: 预算约束
  - valuation_dist: 隐私估值分布

- `pac_mechanism(valuations, epsilon_budgets)`: 实现PAC机制
  - 输入：客户端隐私估值、隐私预算
  - 输出：分配向量q、支付向量p
  - 复杂度：O(N log N)

- `compute_utility(payment, valuation, epsilon)`: 计算客户端效用
  - 公式：u_i = p_i - c(v_i, ε_i) = p_i - v_i * ε_i

- `check_dsic(allocations, payments, valuations)`: 验证DSIC性质
  - 计算平均事后遗憾
  - 返回遗憾值和IR违反率

- `check_ir(payments, valuations, epsilon)`: 验证IR性质
  - 检查所有客户端效用是否非负
```

### 2.2 MFG系统模块 (`mfg_system.py`)

**功能**：求解HJB-FP耦合系统，计算均衡

```markdown
### 主要类和函数

#### MeanFieldGame
- `__init__(state_dim=2, time_horizon=1.0)`: 初始化MFG
  - state_dim: 状态维度（v, ε）
  - time_horizon: 时间范围

- `solve_hjb_fp(initial_distribution, drift_fn, payment_fn)`: 求解HJB-FP系统
  - 输入：初始分布μ_0、漂移函数f、支付函数p̃
  - 输出：值函数Φ*、均衡分布μ*
  - 方法：时间离散化 + 有限差分法

- `compute_mean_field_bid(distribution)`: 计算均衡均值场竞价
  - b_MFG = E_μ[b] = ∫ b(s) dμ(s)
  - 返回标量统计量

- `verify_mfe_approximation(N)`: 验证MFE近似误差
  - 理论误差：ε_N = O(1/√N)
  - 返回实际误差估计

#### HJBSolver
- `solve(value_fn, hamiltonian, boundary_condition)`: 反向求解HJB
  - 使用粘性解理论
  - 返回最优值函数

#### FokkerPlanckSolver
- `solve(initial_dist, drift_fn, diffusion_coeff)`: 正向求解FP
  - 使用Wasserstein梯度流
  - 返回演化分布序列
```

### 2.3 MFG-RegretNet网络模块 (`regretnet.py`)

**功能**：深度学习拍卖求解器

```markdown
### 主要类和函数

#### AuctionHeaderNetwork
- `__init__(input_dim, hidden_dim, output_dim)`: 初始化AHN
  - 输入：[b_1, ..., b_N, b_MFG] ∈ R^{(N+1)×m}
  - 输出：上下文嵌入 C ∈ R^{k×m}

- `forward(bids, mean_field_bid)`: 前向传播
  - 拼接输入：a = [b_1; ...; b_N; b_MFG]
  - 返回共享表示C

#### AllocationNetwork
- `__init__(context_dim, num_items)`: 初始化分配网络
  - 输出：分配矩阵 Z ∈ [0,1]^{N×m}
  - 约束：列向Softmax，∑_i z_ij ≤ 1

- `forward(context)`: 前向传播
  - 返回分配矩阵Z

#### PaymentNetwork
- `__init__(context_dim, num_items)`: 初始化支付网络
  - 输出：支付矩阵 P̂ ∈ [0,1]^{N×m}
  - 约束：Sigmoid激活，确保非负

- `forward(context)`: 前向传播
  - 返回支付矩阵P̂

#### MFGRegretNet
- `__init__(N, m, B, network_config)`: 初始化完整网络
  - 组合AHN + AllocationNet + PaymentNet

- `forward(bids, mean_field_bid)`: 前向传播
  - 返回分配Z、支付p̄

- `compute_regret(bids, mean_field_bid, pga_steps=25)`: 计算MFG感知遗憾
  - 使用投影梯度上升(PGA)估计最大遗憾
  - 关键：b'_MFG = b_MFG + (b'_i - b_i)/N（自洽更新）
  - 返回遗憾向量

- `compute_loss(bids, mean_field_bid, labels=None)`: 计算增强拉格朗日损失
  - L_base = -∑p_i + γ∑[c(v_i,ε_i) - p_i]_+
  - L_AL = L_base + ∑λ_i * rgt_i + (ρ/2)∑rgt_i^2
  - 返回总损失

- `train_step(bids, mean_field_bid, optimizer, lambda_multipliers, rho)`: 单步训练
  - 更新网络参数θ
  - 更新拉格朗日乘子λ_i
  - 返回损失值
```

### 2.4 联邦学习框架模块 (`federated_learning.py`)

**功能**：FL训练与隐私机制集成

```markdown
### 主要类和函数

#### FederatedLearningClient
- `__init__(client_id, local_data, model, privacy_budget)`: 初始化客户端
  - client_id: 客户端标识
  - local_data: 本地数据集
  - model: 本地模型
  - privacy_budget: 隐私预算ε_i

- `local_train(epochs, learning_rate)`: 本地训练
  - 返回本地梯度w_i

- `apply_dp_noise(gradient, epsilon, delta, sensitivity)`: 应用DP噪声
  - 高斯机制：w̃_i = w_i + N(0, σ²I)
  - σ² = 4Δ_f² ln(1/δ) / ε²
  - 返回噪声梯度

#### FederatedLearningServer
- `__init__(global_model, num_clients, budget)`: 初始化服务器
  - global_model: 全局模型
  - num_clients: 客户端数
  - budget: 总隐私预算B

- `aggregate_gradients(client_gradients, weights)`: 聚合梯度
  - w_G = ∑α_i * w̃_i，其中α_i = ε_i^out / ∑ε_j^out
  - 返回全局梯度

- `run_fl_round(clients, auction_mechanism)`: 执行一轮FL
  - 步骤1：客户端本地训练
  - 步骤2：运行隐私拍卖
  - 步骤3：应用DP噪声
  - 步骤4：聚合梯度
  - 步骤5：广播全局模型

#### PAGFLFramework
- `__init__(server, clients, auction_solver)`: 初始化PAG-FL框架
  - server: FL服务器
  - clients: 客户端列表
  - auction_solver: MFG-RegretNet求解器

- `train(num_rounds, eval_freq)`: 训练主循环
  - 循环执行FL轮次
  - 定期评估模型精度
  - 记录拍卖指标

- `evaluate(test_data)`: 评估全局模型
  - 返回测试精度
```

### 2.5 基线方法模块 (`baselines.py`)

**功能**：实现所有对比基线

```markdown
### 主要类

#### PACMechanism
- 实现Algorithm 1（阈值机制）
- 复杂度：O(N log N)

#### VCGAuction
- 实现Vickrey-Clarke-Groves机制
- 复杂度：O(N²)

#### VanillaRegretNet
- RegretNet不含MFG组件
- 输入：仅b，不含b_MFG

#### DMRegretNet
- RegretNet应用于FL模型交易
- 多项目设置

#### MFGPricing
- MFG数据定价（无拍卖）
- 基于均值场博弈的定价

#### CSRAReverseAuction
- CSRA机制（IEEE TIFS 2024）
- 鲁棒反向拍卖
```
